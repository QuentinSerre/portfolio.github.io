Qu'est ce qu'un framework ?

Un Framework signifie “cadre de travail” et permet aux développeurs d'être plus efficaces dans la conception d'applications web, en offrant une architecture et des composants logiciels. Les Framework fonctionnent par langage de programmation et permettent de développer tous types de supports : applications mobiles, logiciels, plateformes web, etc. Il est aussi possible de créer son propre Framework.

Parmi les Framework les plus utilisés nous retrouvons :

  • ASP.NET Core : Framework open-source
  • Node.js : Framework javascript
  • Django : Framework Python
  • Laravel : Framework PHP

L'tilisation de Framework offre plusieurs avantages :

  • Gagner du temps
  • Avoir un cadre de travail clair et organisé
  • Optimiser son travail en équipe
  • Optimiser la sécurité

Les Framework sont donc monnaie courante dans le monde de l'informatique et souvent utilisé pour leur praticité et leur gain de temps durant le travail.
